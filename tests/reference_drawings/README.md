这些文件是归档几何基准。自动审计将固定模板失败、产品图 R、导轨截面
R、截面弧面方向和未裁决规则冲突分别输出；产品图中存在某个 R 不能作为
导轨截面必须使用该 R 的证据。

已确认的生产规则：

- 三图均为正确生产图纸，固定上口采用 2.00 mm。
- 实例 1 为下部 R16.3、上部平面，R 圆心位于上侧；双 R 产品的 R 面必须
  与第一砂轮同侧，所有机台执行同一方向规则。
- 实例 2、3 的成品为单 R 面包型、磨前为方块，导轨截面采用矩形槽。
- 槽宽和导轨厚度继续执行 QG 38012；因此三图历史厚度 `+0.10 mm` 与
  当前 QG `+0.12 mm` 的差异，以及实例 3 的 `12.15/12.14 mm` 差异，
  记录为已批准的规则覆盖，不属于未解决冲突。
- 实例 3 归档槽宽为 12.15 mm；当前 QG 规则按 0.02 mm 公差范围取
  0.04 mm 间隙，结果为 12.14 mm。

审计中只有明确批准的 QG 差异可标记为 `APPROVED_RULE_OVERRIDE`；槽型、
R 方向、固定上口或其他未批准差异仍标记为
`UNRESOLVED_RULE_CONFLICT`。

审计入口：`python3 -m src.reference_dxf_audit`。
