# 三个双规格实例规则审计

## 审计范围

本审计对照以下三类证据：

1. 用户提供的三个归档 DXF；
2. 成型磨前规格与成品规格；
3. `QG 38012《成型磨导轨设计方法》`、当前机台配置和生成结果。

审计仅陈述实测结果和规则冲突。未获得工艺确认前，不以当前目标、旧图纸
或程序现状中的任何一方自动覆盖另一方。

## 可以直接确认的规则

### 双规格必须独立

实例 3 的磨前宽度为 12.2 mm，成品宽度为 12 mm，而参考槽宽为
12.15 mm。该实例证明导轨通过尺寸不能由成品宽度替代，槽基础尺寸必须
由磨前规格及公差计算。

### 产品 R 与导轨截面 R 必须分开

实例 2、3 的产品图中分别存在 R9.6 和 R24.7，但导轨截面均为矩形。
因此，成品 R 是加工目标证据，不等同于导轨截面必须包含该 R。

### 双 R 不能直接生成双弧槽

实例 1 的成品包含 R16.3 和 R14.3，但参考导轨截面只使用 R16.3，且另
一侧为平面。该机型上选取较大成品 R 有实例依据；R 面方向按第一砂轮侧
统一确定。

## 已确认规则

### 槽型和工序状态

成品为单 R 面包型且磨前为方块时，导轨统一采用矩形槽。成品 R 只作为
磨后目标保存，不进入导轨截面。

### 实例 1 弧面方向

实例 1 的主圆弧圆心在型腔上侧，圆弧构成型腔下表面，即“下部 R、上部
平面”。QG 38012 对 379 mm、砂轮“下、上”机型的文字规则也采用该方向。
生产规则确认为：双 R 产品的导轨 R 面与第一砂轮同侧，圆心和平面在对
侧。本机第一砂轮在下，因此生成下部 R、上部平面。该规则适用于所有机台。

### 第一砂轮侧定义

实例输入和机台配置现在统一为：

```text
wheel_sequence = ["下", "上"]
first_wheel_side = "lower"
```

`wheel_sequence` 第一项作为第一砂轮方向；`first_wheel_side` 必须与其
一致。程序同时检查 R 面与第一砂轮同侧、圆心与第一砂轮对侧，字段冲突时
禁止 release。

### 导轨厚度间隙

三个参考实例均为磨前名义厚度加 0.10 mm；工艺确认继续执行 QG 38012，
方块产品按磨前厚度中值加 0.12 mm。该差异作为已批准规则覆盖保留。

### 中心上口

三个参考实例均为 2.00 mm，已写入该机型配置。

### 实例 3 槽宽

按当前 QG 公式，实例 3 的磨前宽度中值为 12.10 mm，0.04 mm 间隙得到
12.14 mm；参考图为 12.15 mm。实例 1、2 与当前公式一致，因此不能仅凭
实例 3 修改全局公式。工艺确认继续采用 QG 结果 12.14 mm，并把参考图
12.15 mm 记录为已批准规则覆盖。

## 审计门禁

参考对比结果使用以下状态：

```text
MATCH
APPROVED_RULE_OVERRIDE
UNRESOLVED_RULE_CONFLICT
FAIL
```

当前三个实例的槽型、方向和固定尺寸均已匹配。QG 与历史图纸的数值差异
标记为 `APPROVED_RULE_OVERRIDE`，最终状态为 `PASS`。产品图中存在某个
R 仍只能作为产品目标轮廓证据，不能让导轨截面 R 校验自动通过。

## 工艺确认记录

1. 三份参考图均为有效生产标准；
2. 双 R 的导轨 R 面跟随第一砂轮方向；
3. 单 R 面包型成品加方块磨前采用矩形槽；
4. 槽宽和导轨厚度继续执行 QG 38012；
5. 三头机单导轨（下上）中心上口采用参考图固定值 2.00 mm。

以上选择已写入机台配置、集中决策、几何生成、自动测试和参考对比逻辑，
所有尺寸同时修改真实几何与 DIMENSION 定义点。
